
## Run the app

```bash
# development
- `npm run start`

# watch mode
- `npm run start:dev`

# production mode
- `npm run start:prod`
```
## Testing the Running App
- Go to browser - localhost:3030/api
- make requests here and get token
- use token to make curl request e.g. to get `users` (testing `auth`)
## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
